【项目信息】
项目名称 : dbunit3
中文名称 : 数据库 单元测试
项目版本 : version 1.0
主要技术 : dbunit

项目描述 : 数据库/dao层 测试框架
1)dbunit 通过 备份 - 录入数据 - 测试 - 还原 4个流程来测试dao层，测试之前先进行清场备份，然后准备一些简单的、容易的数据进行
测试，增删改可以利用Assertion.assertEquals()方法，用表的整体进行比较，因此测试准备的数据一般都很少，不会太多;
查询方法可以通过具体的字段来展开比较.
2) 单张表的测试比较简单，数据量相对少，方法设计上整体可预测，但是多张表情况比较多，方法设计上相对复杂.
3) 在项目的开发数据库中，前期测试dao层的时候，数据表的记录一般不会很多，所以整个数据还原不会太卡.

项目备注 : dbunit version 3.x
